### Запуск

```
php artisan migrate
php artisan serve
npm run dev
```

### Что сделано

1. Пользователь может управлять только своими заметками (проверка user_id) - *DONE*
2. Все пользовательские данные экранируются при выводе  - *DONE*
3. Prepared statements для всех запросов к БД  - *DONE* (через Laravel, у которого под капотом параметризованные запросы, но не через сырые запросы)
4. CSRF-токен в формах  - *DONE*

5. Карточки заметок в сетке (grid), а не таблица - *Я бы доделал*
6. Цветная полоска сверху карточки соответствует color заметки  - *DONE*
7. Иконка булавки для закрепленных заметок  - *DONE*
8. Кнопка pin/unpin через Alpine.js + fetch (без перезагрузки страницы)  - *DONE*
9. Поисковая строка с фильтрацией  - *DONE*
10. Пагинация  - *DONE* (я поставил 12, а не 20)
11. Весь CRUD и БД - *DONE*
